# Design System - Golf Assistant

## ビジュアルテーマ

- **方向性**: クリーン・機能的・アウトドア耐性（屋外でも視認性が高い）
- **参考**: Arccos Golf（ゴルフ特化UI）、Strava（スポーツ系の実用性）、Linear（ミニマルな操作性）
- **モード**: ライト（メイン画面）/ ダーク（プレー中画面）
- **原則**: マニュアル不要で操作できる直感性。ラベル・アイコン・色で意味が伝わること

## タイポグラフィ

- **フォント**: Geist Sans（欧文）、system-ui（和文フォールバック）
- **ページタイトル**: 24px / bold / 行間1.3
- **セクション見出し**: 18px / semibold / 行間1.4
- **本文**: 16px（モバイル基準） / regular / 行間1.6
- **ラベル**: 14px / medium / 行間1.5
- **キャプション**: 12px / regular / 行間1.5
- **ボタン**: 14px / medium
- **プレー中数値**: 20px-36px / bold（スコア等の大きい数字）

## スペーシング

- **基本単位**: 4px
- **セクション間**: 24px-32px
- **カード内パディング**: 12px（sm）, 16px（md）, 24px（lg）
- **要素間**: 8px-12px
- **ページ水平パディング**: 16px（モバイル）

## 回避パターン（AI臭さの排除）

以下のパターンは使用禁止:

- 過剰なグラデーション背景（特にヒーローセクション）
- 無意味な装飾的グリッドやドットパターン
- テンプレート感のあるカード3列均等配置
- 全要素に同一の角丸・影の適用
- ストック写真的なプレースホルダー画像
- 「Welcome to...」「Get Started」「Your Journey Begins」等の定型コピー
- 過剰なアイコン装飾
- 根拠のない紫〜青のグラデーションCTA
- 意味のないマイクロアニメーション

## レイアウト原則

- **モバイルファースト**: プレー中は片手操作を前提
- **タッチターゲット**: 最小48px（屋外・手袋着用を想定）
- **情報密度**: 1画面に詰め込みすぎない。スクロールで段階的に開示
- **視覚的階層**: サイズ・色・太さで優先度を明示
- **直感的操作**: アイコン+ラベルの併用、色による状態表現、操作結果の即座フィードバック

## プレー中画面の特別ルール

- ダークモード固定（屋外での目の疲労軽減）
- 数値は大きく表示（スコア、残りヤード等）
- 操作ボタンは画面下部に集約（片手の親指で届く範囲）
- 状態変化は色で即座にフィードバック（emerald/amber/rose）
